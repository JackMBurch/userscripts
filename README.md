# userscripts

A collection of browser userscripts maintained by [JackMBurch](https://greasyfork.org/en/users/JackMBurch) and published on [Greasy Fork](https://greasyfork.org/).

Each script lives in its own directory with a `.user.js` file and a README describing what it does.

## Scripts

| Script                                              | Description                                                                                                      |
| --------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| [whatsapp-hide-chat-list](whatsapp-hide-chat-list/) | [WhatsApp Collapsible Sidebars](https://greasyfork.org/en/scripts/584118) — collapse the icon rail and chat list |

## Installing

Install scripts from their Greasy Fork pages, or install a `.user.js` file directly with a userscript manager such as [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/).
